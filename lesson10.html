CSS Transform

Translate (x,y)

Translate

Translateсдвигает элемент на новое место, перемещая относттельно изначального положения объекта вправо и вниз, используя координаты х и у

Scale (x,y)

Scale

Scaleмасштабирует элементы, делаяих больше или меньше. по умолчанию=1, реальный размер. х-ширина и у-высота. отрицательные значения отражают зеркально.

Rotate (x,y)

Rotate

Rotateповорачивает элементы на заданное кво градусов, отриц.значение-против часовой.

Skew (x,y)

Skew

Skewдеформирует(искажает) стороны объекта относительно координатных осей

Matrix (a, c, b, d, x, y)

Matrix (a, c, b, d, x, y)

Matrixобъединяет несколько функций 2Д транф в одной. указывается без ед.изм.
